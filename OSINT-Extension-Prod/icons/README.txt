Please add the following icon files to this directory before using the extension:

1. icon16.png (16x16 pixels)
2. icon48.png (48x48 pixels)
3. icon128.png (128x128 pixels)

These icons should represent your extension in the Chrome browser interface. You can create these icons using any image editing software like GIMP, Photoshop, or use online icon generators.

Recommended icon guidelines:
- Use simple, recognizable designs
- Ensure good visibility at small sizes
- Use transparent backgrounds (PNG format)
- Follow Chrome's icon design guidelines
- Test icons in both light and dark browser themes

You can find icon design resources and guidelines at:
https://developer.chrome.com/docs/webstore/images/
