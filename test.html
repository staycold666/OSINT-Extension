Expected Behavior

  • IP Addresses: Should create a BLUE tab group titled "OSINT - IP Lookup" with 11 tabs (AbuseIPDB, VirusTotal, IPVoid, etc.)
  • SHA Hashes: Should create a RED tab group titled "OSINT - HASH Lookup" with 7 tabs (VirusTotal, Malware Bazaar, etc.)
  • Domain Names: Should create a GREEN tab group titled "OSINT - DOMAIN Lookup" with 8 tabs (VirusTotal, URLScan, etc.)
  • All groups: Should start in collapsed state
  • Fallback: If Tab Groups API is not available, should open tabs normally without grouping

Installation Instructions

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (toggle in top right)
  3. Click "Load unpacked"
  4. Select the OSINT-Extension folder
  5. The extension should now appear in your extensions list
  6. Return to this page and test the right-click functionality
